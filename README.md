# GoTech Data Engineering Challenge

## Overview

This repository contains the solution for the GoTech Data Engineering Bootcamp technical challenge.

The objective is to design and implement an end-to-end data pipeline that ingests raw CSV data, performs data quality validation, builds Bronze, Silver and Gold layers, orchestrates the workflow using Apache Airflow, exports analytical datasets to Parquet and generates business insights.

## Technology Stack

- Python
- PostgreSQL
- Docker
- Apache Airflow
- SQL
- Pandas
- Jupyter Notebook
- Parquet

  Docker Desktop como requisito previo del sistema

## Project Structure

The project follows a layered data engineering architecture:

- Bronze Layer
- Silver Layer
- Gold Layer

Each layer is designed to ensure data quality, traceability and reproducibility.
